Getting Started
===============

Installation
------------

Using Poetry (recommended)::

    pip install poetry
    poetry install
    poetry run python run.py

Using pip::

    pip install openfootmanager
    openfootmanager

Quick Start
-----------

1. Launch the game
2. Click "New Game" to start a career
3. Enter your manager name
4. Select a league and club
5. Manage your team through the season!

Debug Mode
----------

For testing and development, use "Start Debug Mode" from the home screen.
This gives direct access to all game systems without needing to progress through a career.
